# 🤖 AI Assistant Project

> Um assistente inteligente com suporte a voz, integração com APIs e foco em automação e interação em tempo real.

---

## 📌 Sobre o Projeto

Este projeto é uma aplicação de Inteligência Artificial desenvolvida em Python que combina múltiplas funcionalidades em um único sistema.

A proposta é criar um assistente capaz de:
- Interagir com usuários por voz  
- Responder perguntas técnicas  
- Integrar dados de serviços externos  
- Expandir facilmente com novas funcionalidades  

---

## ✨ Funcionalidades

### 🎤 Interação por Voz
- Entrada em chamadas com usuários  
- Respostas por voz em tempo real (Text-to-Speech)  
- Suporte a comandos falados  

---

### 💻 Assistente Técnico
- Responde dúvidas sobre tecnologia  
- Auxilia em programação, sistemas e conceitos computacionais  
- Pode ser usado como suporte rápido para desenvolvedores  

---

### 🎵 Integração com Spotify
- Busca informações de:
  - Músicas  
  - Artistas  
  - Álbuns  
- Utiliza a API oficial do Spotify  

---

### 🎮 Integração com Jogos
- Consulta dados de jogos via APIs externas  
- Pode incluir estatísticas, informações e dados dinâmicos
